# Desk Tour活动

## 活动目的
- 为了彰显公司积极向上正能量的工作面貌
- 促进伙伴们保持个人办公区整洁 提高工作效率
- 加强部门间伙伴们的交流沟通

## 活动形式
- 以部门为单位派选代表参加
- 以图文或视频方式进行本人办公桌面介绍
- 风格轻松愉快、严肃认真均可


# 会议室命名

## 活动目的
- 为了给伙伴提供轻松愉悦的工作环境
- 提高会议室使用效率
- 彰显公司新一代企业文化素养

## 活动形式
- 分别为5个会议室进行个性化命名
- 正能量成语类命名 (独占鳌头 口碑载道 等)
- 高分电影类命名 (阿甘正传 等)
- 励志文学类命名 (追风筝的人 等)
- 积极象征类动植物命名 (向日葵 蜗牛 等)
